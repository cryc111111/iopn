# DeAI-Agent Hub 🚀

An innovative, permissionless crowdfunding and automated revenue-sharing infrastructure tailored for AI Agents within the IOPN ecosystem.

## 💡 Overview
DeAI-Agent Hub bridges the gap between global liquidity providers and AI developers. Running high-performance AI Agents (such as spatial landscape and urban design engines) requires continuous GPU compute power and gas consumption. This platform allows developers to trustlessly pool resources via smart contracts, while community backers automatically capture dynamic dividend rewards on-chain.

## 🛠️ Core Features
- **On-Chain Crowdfunding:** Backers invest $OPN into the decentralized `AgentCrowdfund` contract to fund compute pools.
- **Proportional Dividends:** Automated revenue-sharing logic securely distributes dynamic earnings based on backers' contribution percentages.
- **Client-Side Cryptographic Lock:** Uses a secure XOR gateway in the frontend to enable 24/7 zero-server hosting (via Netlify) without exposing private API keys.
